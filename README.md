# calculadoraImc
App em Kotlin com Jetpack Compose que calcula o IMC a partir da altura e peso do usuário.

## Funcionalidades
- Cálculo de IMC  
- Classificação (abaixo, ideal, sobrepeso, obesidade)  
- Exibição do resultado na tela  

## Tecnologias
- Kotlin  
- Jetpack Compose  
- Material 3
